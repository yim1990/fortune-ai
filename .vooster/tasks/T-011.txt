# 로컬 개발환경 통합 실행/종료 스크립트 제공

**Task ID:** T-011
**Status:** BACKLOG
**Importance:** SHOULD
**Complexity:** 4/10
**Urgency:** 7/10
**Dependencies:** T-008

## Description

설명: docker compose 기반 통합 실행/종료 가이드를 README에 문서화하고, 선택적으로 편의 스크립트를 제공한다.
요구사항: 1) 실행: docker compose up -d --build 2) 종료: docker compose down -v 3) 로그 확인: docker compose logs -f web|php-api 4) 선택: scripts/dev-up.sh, scripts/dev-down.sh 제공 및 실행 권한 부여
구현 상세: - 루트 README에 명령어와 기대 결과, 트러블슈팅(Q&A) 추가 - 스크립트는 POSIX 셸로 작성, Windows 사용자는 직접 명령 실행 안내 - 컨테이너 상태 점검(docker compose ps) 가이드 포함
의사코드: ```bash
# scripts/dev-up.sh
#!/usr/bin/env bash
set -euo pipefail
docker compose up -d --build
# scripts/dev-down.sh
#!/usr/bin/env bash
set -euo pipefail
docker compose down -v
```
테스트 전략: - chmod +x scripts/*.sh 후 실행 - up → ps 상태 healthy 확인, down 후 네트워크/볼륨 제거 확인 - README 단계별 지침에 따라 신규 개발자가 10분 내 환경 재현 가능한지 점검
